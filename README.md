## Installation

Using poetry:

```sh
$ poetry add quantaq-cli
```

Using pip:

```
$ pip install quantaq-cli
```

Or, you can clone the repository and install locally:

```sh
$ poetry install
```

## Testing

```sh
$ poetry run pytest tests
```
